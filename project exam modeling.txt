The following script uses scipy.integrate.solve_ivp, which is the modern standard for solving initial value problems in Python.
```python
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

# 1. Define Parameters
m = 1.0    # mass (kg)
c = 0.5    # damping coefficient (Ns/m)
k = 10.0   # spring constant (N/m)

# 2. Define the system of ODEs
def damped_oscillator(t, y):
    x, v = y
    dxdt = v
    dvdt = -(c/m)*v - (k/m)*x
    return [dxdt, dvdt]

# 3. Set Initial Conditions and Time Span
# x(0) = 1.0 m, v(0) = 0.0 m/s
y0 = [1.0, 0.0]
t_span = (0, 20)
t_eval = np.linspace(0, 20, 1000) # Points where we want the solution

# 4. Solve the ODE
sol = solve_ivp(damped_oscillator, t_span, y0, t_eval=t_eval)

# 5. Plotting
plt.figure(figsize=(10, 6))
plt.plot(sol.t, sol.y[0], 'b-', linewidth=2, label='Displacement (x)')
plt.axhline(0, color='black', linestyle='--', alpha=0.3) # Equilibrium line
plt.title('Damped Harmonic Oscillator Motion')
plt.xlabel('Time (s)')
plt.ylabel('Displacement (m)')
plt.grid(True, linestyle=':', alpha=0.6)
plt.legend()
plt.show()

```
### 3. Understanding the Physics
In your specific simulation (with c=0.5, m=1, k=10), the system is **underdamped**. This is determined by the damping ratio \zeta (zeta):
 * If \zeta < 1: The system oscillates with a decaying amplitude (Underdamped).
 * If \zeta = 1: The system returns to equilibrium as quickly as possible without oscillating (Critically Damped).
 * If \zeta > 1: The system returns to equilibrium slowly without oscillating (Overdamped).
For your values, \zeta = \frac{0.5}{2\sqrt{10}} \approx 0.079, which confirms the oscillating "decay" you will see in the generated plot.

This is a classic problem in stochastic processes, often used to model everything from gas diffusion to stock market fluctuations. Below is a clean, efficient Python implementation using numpy for the math and matplotlib for the visualization.
### Python Simulation: 2D Random Walk
```python
import numpy as np
import matplotlib.pyplot as plt

def simulate_random_walk(num_steps):
    # Initialize arrays to store coordinates (starting at 0,0)
    x = np.zeros(num_steps + 1)
    y = np.zeros(num_steps + 1)
    
    # Define possible moves: Up, Down, Left, Right
    # Each row is a vector: [dx, dy]
    directions = np.array([[0, 1], [0, -1], [-1, 0], [1, 0]])
    
    for i in range(1, num_steps + 1):
        # Randomly select one of the four directions
        step = directions[np.random.randint(0, 4)]
        
        # Update coordinates
        x[i] = x[i-1] + step[0]
        y[i] = y[i-1] + step[1]
        
    return x, y

# Parameters
n = 1000
x_coords, y_coords = simulate_random_walk(n)

# Calculate Final Displacement (Euclidean distance)
final_displacement = np.sqrt(x_coords[-1]**2 + y_coords[-1]**2)

# Plotting the Trajectory
plt.figure(figsize=(8, 8))
plt.plot(x_coords, y_coords, label='Path', lw=1, alpha=0.8)
plt.scatter(0, 0, color='green', label='Start (0,0)', zorder=5)
plt.scatter(x_coords[-1], y_coords[-1], color='red', label='End', zorder=5)

# Formatting
plt.title(f'2D Random Walk Simulation ({n} Steps)')
plt.xlabel('X Coordinate')
plt.ylabel('Y Coordinate')
plt.legend()
plt.grid(True, linestyle='--', alpha=0.6)
plt.axis('equal')  # Ensures 1 unit on x is same as 1 unit on y

# Display result
print(f"Final Position: ({x_coords[-1]}, {y_coords[-1]})")
print(f"Final Displacement from Origin: {final_displacement:.2f} units")

plt.show()

```
